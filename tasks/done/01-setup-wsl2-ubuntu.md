# Task: Set Up WSL2 Ubuntu Environment

**Status: Done**

## Description
Enable and configure WSL2 with Ubuntu on your Windows system to provide a Linux environment for GPU-accelerated machine learning.

## Steps
- Enable Windows features: Virtual Machine Platform and Windows Subsystem for Linux
- Install Ubuntu from Microsoft Store or PowerShell
- Set WSL2 as default version
- Launch Ubuntu and create user account
- Update Ubuntu system packages
